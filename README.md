# Ghibli Proxy

Proxy para consumir e manipular dados da API do Studio Ghibli.

## Instalação

```bash
git clone https://github.com/seu-usuario/ghibli-proxy.git
cd ghibli-proxy
npm install
```

## Uso

Inicie o servidor com `npm start` e acesse em `http://localhost:3000`.
